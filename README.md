# Klasifikasi Sentimen dan Hubungan Karakteristik Teks terhadap Tingkat Engagement Tweet pada Layanan Shopee

Repository ini berisi kode sumber dan berkas pendukung penelitian skripsi dengan judul:

**“Klasifikasi Sentimen dan Hubungan Karakteristik Teks terhadap Tingkat Engagement Tweet pada Layanan Shopee”**

Penelitian ini membahas penerapan metode klasifikasi sentimen serta analisis karakteristik teks terhadap tingkat engagement pada data Twitter (X).
